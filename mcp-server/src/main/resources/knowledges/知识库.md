# 知识库
## Fast Request 知识库

### 简介
Fast Request是一个类似于 Postman 的 IDEA 插件。它是一个强大的 restful api 工具包插件，可以根据已有的方法帮助您快速生成 url 和 params。 Restful Fast Request = API调试工具 + API管理工具 + API搜索工具。 它有一个漂亮的界面来完成请求、检查服务器响应、存储你的 api 请求和导出 api 请求。插件帮助你在 IDEA 界面内更快更高效得调试你的 API。

对比于 HTTP Client，Fast Request 不仅拥有 HTTP Client 内置的功能，还提供了友好易懂直观的界面，让使用者调试 API 的时候能够更加方便、简捷。同时各种类型参数也提供了不同的定制方式，更加灵活。而且集成了搜索、Postman 集成，Swagger 支持等额外的功能。

官网地址：[https://api-buddy.cn/](https://api-buddy.cn/)

### 安装
IDEA 插件市场搜索 Restful Fast Request 并安装

## Postman 知识库

### 简介
Postman 是一个非常流行的 API 测试工具，支持 RESTful API 和 SOAP API。它提供了一个用户友好的界面，允许开发人员和测试人员轻松地发送 HTTP 请求、查看响应、管理 API 集合以及进行自动化测试。

### 主要功能
1. **发送请求**：支持各种 HTTP 方法（GET、POST、PUT、DELETE 等），可以自定义请求头、参数和请求体。
2. **查看响应**：显示响应状态码、响应时间和响应体，支持多种格式（JSON、XML、HTML 等）。
3. **管理集合**：可以将相关的 API 请求组织成集合，方便管理和共享。
4. **环境变量**：支持环境变量的使用，方便在不同环境（开发、测试、生产）之间切换。
5. **自动化测试**：支持编写测试脚本，进行自动化测试和断言。
6. **团队协作**：支持团队成员之间共享 API 集合和环境设置。
7. **集成**：可以与 CI/CD 工具集成，实现持续集成和持续交付。

### 安装
1. 访问 Postman 官网：[https://www.postman.com/](https://www.postman.com/)
2. 下载适用于你操作系统的版本（Windows、macOS、Linux）。
3. 安装并启动 Postman 应用程序。
